History of the font

Recursive is a variable font that was created by Stephen Nixon. The font is inspired by New York City sign painting. He started this project as a a thesis project and Google commissioned him to complete it. Recursive was made with variable fonts in mind and the ability for fonts to be more versatile. Casual sign painting is a type of lettering that was made to be easy to learn, and each letter is only made with a few strokes. This makes the font more practical because it allows room for imperfections. Nixon is from New York City, and casual lettering is used for the signs of many of the businesses in the city. This typeface was designed to have personality originating from hand painted signs. In order for the font to have many use cases, it had to be able to look more simple if needed. One of the axises is called casual, which allows for the font to look more like a “normal” sans serif.
